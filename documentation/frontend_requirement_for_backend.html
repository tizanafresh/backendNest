🔌 Tizanas Fresh

Requerimientos de APIs para Backend

Especificaciones para el Proyecto NestJS - Versión 1.0

📋 Introducción

Este documento especifica todas las APIs necesarias para el backend que soportará la aplicación móvil React Native de Tizanas Fresh. Las APIs están organizadas por módulos funcionales y incluyen detalles completos de endpoints, parámetros, respuestas y prioridades de implementación.

Nota: Todas las APIs deben implementar autenticación JWT excepto las marcadas como públicas. Los códigos de respuesta HTTP estándar deben ser utilizados (200, 201, 400, 401, 403, 404, 500).
